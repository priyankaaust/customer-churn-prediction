# Customer Churn Prediction

## Key Learning
This project demonstrates how raw data can be transformed into actionable insights using machine learning, even with minimal preprocessing.

## Problem
Businesses struggle to identify customers likely to leave, leading to revenue loss.

## Dataset
This project uses the Telco Customer Churn dataset:
https://www.kaggle.com/blastchar/telco-customer-churn

## Data Instructions
Download the dataset from Kaggle and place it in the project folder before running the code.

## Approach
- Cleaned and preprocessed dataset
- Encoded categorical variables
- Built a Random Forest model
- Evaluated performance using accuracy

## Tools Used
Python, pandas, scikit-learn

## Results
- Achieved ~80% accuracy
- Demonstrated predictive insights for customer retention

## How to Run
1. Install required libraries:
   pip install pandas scikit-learn

2. Run the script:
   python churn_model.py
